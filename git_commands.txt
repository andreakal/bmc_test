git log : look at the log of commits
git status : see the status of the repo
git clone [repo-url]: clone a repository
git show : Show the last commit
git diff : see changes
git branch : see available branches
git branch [branch-name] : create a new branch
git checkout [branch-name] : checkout a branch
git checkout [commit-number] : checkout a specific commit
git push --set-upstream origin [branch-name] : add your branch to the upstream repo
git merge [branch-name] : merge changes from branch-name into the branch that you have currently checked out.
